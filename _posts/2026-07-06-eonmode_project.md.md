# Eon Mode 재구축 프로젝트

구분: 실무
기간: 2021년 3월 1일 → 2021년 4월 30일
기술 스택: SQL
상태: 완료
설명: Vertica Eon Mode 기반 환경 설계 및 120TB 실시간 무중단 마이그레이션 수행

### 프로젝트 개요

---

기존 시스템 수가 많고 장비가 노후화됨에 따라, 장비 수를 최소화하면서도 데이터 저장소의 유연한 확장이 가능한 새로운 아키텍처가 필요했습니다.

이에 Vertica Eon Mode 기반 환경을 설계하고, 실시간 무중단 마이그레이션을 수행(120TB)했습니다.

### 활용 기술

---

- SQL
- Shell Script

### 주요 활동

---

1. 신규 Eon Mode 환경 설계 및 테이블 파티셔닝
- Vertica Eon Mode 기반 새로운 DB 환경을 설계
- 데이터 볼륨이 높은 테이블에 파티셔닝 전략을 적용하여 성능을 최적화
1. 실시간 무중단 마이그레이션 수행
- Vertica DBMS 내부 복제 함수를 활용해 실시간 데이터 마이그레이션을 수행
- 시스템 운영 중단 없이 데이터 이전을 완료하여 무중단 마이그레이션을 구현
- 기존 운영 서버 수를 1/3로 줄여 운영비 절감에 기여
- 쿼리 성능 향상 및 리소스 낭비 최소화로 분석 작업 효율성을 극대화

### 성과

---

- 대용량 환경에서 최적화된 아키텍처 설계 및 파티셔닝 적용 경험을 확보
- 실시간 마이그레이션으로 안정적 데이터 운영과 시스템 전환 역량을 강화
